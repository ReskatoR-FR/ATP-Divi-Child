== Description ==
Create a Divi child theme with one click.

The created child theme is fully compliant with the latest specifications of the WordPress Codex.

You can define the name of Divi child theme, the author, the author URL, the version number and select an image that will be customized for your child theme.

== Installation ==
How to install the ATP Divi Child Plugin

= VIA WordPress =
1. After your download log into your WordPress admin interface
2. After login click on *Plugins* from the left hand menu
3. Click on *Add new*
4. Click on *Upload*
5. Browse to the directory you downloaded the plugin to and click *Install Now*
6. After WordPress has finished unpacking the file click on *Activate Plugin*
7. After the plugin has been activated you will notice a new Appearance submenu item labelled ATP Divi Child
8. Done. Start creating your child theme!

= VIA FTP =
1. After your download, unzip ATP Divi Child from your download .zip
2. Open your FTP client
3. Browse to /your_wp_instalation/wp-content/plugins/ on your hosting server
4. Uplaod rr-atp-divi-child directory into this directory
5. Go to your wordpress admin panel
6. Go to plugins and activate RR ATP Divi Child
7. After the plugin has been activated you will notice a new Appearance submenu item labelled ATP Divi Child
8. Done. Start creating your child theme!

== Frequently Asked Questions ==

= How much does this plugin do? =
This plugin creates a child theme from  for you. It doesn't do anything to the database, it doesn't do anything to your new child theme after you've created it. It only gives you buttons to create a child theme.
